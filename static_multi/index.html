系統運作原理 (RAG System Flow)

1

文件拆解與切分

自動掃描網頁內容 docx 資料夾，保留段落與表格，將長文切分為 500-800 字的語意區塊

2

向量化與索引

3

語意向量檢索

將提問即時轉為向量，利用 NumPy 在記憶體快速進行餘弦相似度運算，匹配出 Top-4 最相關的手冊內容

4

脈絡強化生成

將檢索內容組裝為脈絡 Context 餵給本地 qwen2.5 7B 模型，以 SSE 流式生成具備精準出處的回答

關於本系統

本多知識 RAG 系統由 Antigravity 代理程式自動化建置，可擴展性極佳。您可以直接將任何其他的網頁內容導出為 `.docx` 格式，置於 `webpage_docs/` 資料夾下，並執行資料重構腳本，即可快速將多樣化的知識庫融入系統，進行智能對答與檢索！
